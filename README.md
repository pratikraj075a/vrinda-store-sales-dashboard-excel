# Vrinda Store Sales Dashboard (Excel)

## Project Overview
This project analyzes Vrinda Store sales data for 2022 using Microsoft Excel. The dashboard provides insights into sales performance, customer demographics, order status, and sales channels.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## Dashboard Features
- Monthly Sales & Orders Trend
- Sales by Gender
- Order Status Analysis
- Top Revenue Generating States
- Age Group vs Gender Analysis
- Orders by Sales Channel
- Interactive Filters (Month, Channel, Category)

## Key Insights
- Women contribute approximately 64% of total sales.
- Maharashtra generated the highest revenue (2.98M).
- Adult women are the most valuable customer segment.
- Amazon, Myntra, and Flipkart contribute around 80% of total orders.
- 92% of orders were successfully delivered.
